Reading Pad

Is a free software for reading different types of e-Books. This software mainly supports the following types of e-Books: 

PDB / PRC - container format for code databases in Palm OS, Garnet OS and Access Linux Platform. 
PDF - Portable Document Format 
TXT / TEXT - Simple text file 
CBR / CBZ / RAR - Comic book files
